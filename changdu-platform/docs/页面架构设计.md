# 常读分销平台 — 页面架构设计

> **版本**: v1.0
> **更新日期**: 2026-02-23
> **依据**: 飞书文档《常读分销平台国内网文/短剧入驻指南》第三章、第四章
> **用途**: 供前端开发参考的页面蓝图，定义所有页面、路由、内容及跳转关系

---

## 三、布局模板

### 3.1 PublicLayout（公开页面）

```
┌──────────────────────────────────────────┐
│  Header: Logo | [登录/注册] [立即入驻→]  │
├──────────────────────────────────────────┤
│                                          │
│              页面内容区                    │
│                                          │
├──────────────────────────────────────────┤
│  Footer: 版权 · 备案号 · 联系方式         │
└──────────────────────────────────────────┘
```

- **Header**: 常读分销平台 Logo（左）、登录/注册按钮 + 立即入驻按钮（右）
- **Footer**: 版权信息、ICP 备案号、联系方式
- 适用页面：落地页 `/`、登录注册 `/login`

### 3.2 OnboardingLayout（入驻流程）

```
┌──────────────────────────────────────────┐
│  简化 Header: Logo          [退出登录]    │
├──────────────────────────────────────────┤
│  步骤进度条: ① 信息录入 → ② 审核 →       │
│              ③ 合同签署 → ④ 资质授权      │
├──────────────────────────────────────────┤
│                                          │
│              表单/状态内容区               │
│                                          │
├──────────────────────────────────────────┤
│  [在线客服悬浮按钮]                       │
└──────────────────────────────────────────┘
```

- **简化 Header**: 仅 Logo + 退出登录，无导航菜单
- **步骤进度条**: 四步指示（信息录入 → 审核 → 合同签署 → 资质授权），高亮当前步骤
- **在线客服**: 右下角悬浮按钮，所有入驻页面通用
- 适用页面：`/apply-entry`、`/apply-entry/review`、`/apply-entry/contract`、`/apply-entry/qualification`

### 3.3 DashboardLayout（工作台）

```
┌──────────────────────────────────────────┐
│  TopNav: Logo | 搜索 | 通知 | 头像/账号   │
├─────────┬────────────────────────────────┤
│         │                                │
│ Sidebar │         内容区                  │
│         │                                │
│ 工作台首页│                                │
│ 分销中心 │                                │
│  ├ 短剧  │                                │
│  ├ 网文  │                                │
│  ├ 推广链│                                │
│  └ 回传  │                                │
│ 数据中心 │                                │
│  ├ 应用  │                                │
│  ├ 推广  │                                │
│  ├ 订单  │                                │
│  └ 用户  │                                │
│ 账号管理 │                                │
│ 应用设置 │                                │
│ 结算中心 │                                │
│ 企业信息 │                                │
│         │                                │
├─────────┴────────────────────────────────┤
│  [在线客服悬浮按钮 - 仅企业信息页显示]      │
└──────────────────────────────────────────┘
```

- **TopNav**: Logo（左）、全局搜索、消息通知、用户头像+下拉菜单（右）
- **Sidebar**: 可折叠左侧导航，按功能分组，根据角色权限动态显示/隐藏菜单项
- **内容区**: 顶部面包屑 + 页面标题 + 主内容
- **在线客服**: 仅在企业信息页 `/dashboard/profile` 显示
- 适用页面：所有 `/dashboard/*` 路由

## 四、页面详细设计

### 4.1 公开页面

#### 落地页 `/`

> 现有页面，已实现 Header + Hero 区域

| 区域 | 内容 |
|------|------|
| Header | Logo、登录/注册按钮、立即入驻按钮 |
| Hero | 标题「打造优质网文短剧 开拓分销商业新局」、副标题描述多平台支持、CTA「立即入驻」按钮 |
| Footer | 版权信息 |

**交互**:
- 点击「立即入驻」→ 未登录跳 `/login`，已登录跳 `/apply-entry`
- 点击「登录/注册」→ `/login`

---

#### 登录注册 `/login`

| 区域 | 内容 |
|------|------|
| 登录表单 | 手机号/邮箱 + 密码登录 |
| 注册表单 | 手机号 + 短信验证码 + 设置密码 |
| 第三方登录 | 巨量引擎账号 OAuth 登录按钮 |
| 切换 | 登录/注册 Tab 切换 |

**交互**:
- 登录成功后判断入驻状态：
  - 已入驻 → `/dashboard`
  - 未入驻 → `/apply-entry`
  - 入驻审核中 → `/apply-entry/review`
  - 已通过待签约 → `/apply-entry/contract`
  - 已签约待授权 → `/apply-entry/qualification`

---

### 4.2 入驻流程（对应飞书文档第三章）

#### 信息录入 `/apply-entry`

六步表单，顶部步骤指示子条（1→6），每步独立卡片，支持前进/后退。

| 步骤 | 标题 | 表单字段 | 字段类型 |
|------|------|---------|---------|
| 1 | 集团基本信息 | 集团名称 | 文本输入 |
| | | 集团简介 | 多行文本 |
| 2 | 团队规模 | 团队人数 | 数字选择（下拉） |
| | | 团队描述 | 多行文本（投手/编辑人数、团队成熟度） |
| 3 | 主营业务 | 业务类型 | 多选复选框（网文、短剧、其他） |
| | | 大盘收入 | 数字选择/输入（区间选择） |
| 4 | 分销经验 | 是否从事过分销业务 | 单选（是/否） |
| | | 主要对接平台 | 多选（番茄、七猫、疯读等） |
| | | 广告投放渠道 | 多选（巨量引擎、腾讯广告、百度等） |
| | | 擅长投放类型 | 多选（短剧、网文、其他） |
| 5 | 意向业务 | 意向内容类型 | 多选（付费短剧、付费网文） |
| | | 意向合作平台 | 多选（常读旗下各平台） |
| 6 | 联系方式 | 手机号 | 文本输入 + 获取验证码按钮 |
| | | 短信验证码 | 文本输入（6位） |
| | | 联系人姓名 | 文本输入 |
| | | 邮箱 | 文本输入（email 格式校验） |

**交互**:
- 每步表单校验通过后可进入下一步，支持返回上一步修改
- 第 6 步提交 → 跳转 `/apply-entry/review`
- 被驳回后从 `/apply-entry/review` 返回此页，回填已有数据

---

#### 审核状态 `/apply-entry/review`

三种状态，对应不同 UI：

| 状态 | 显示内容 | 操作 |
|------|---------|------|
| **审核中** | 状态图标（时钟）、提示文案「您的入驻申请正在审核中，请耐心等待」、预计审核时间 | 无操作，可刷新查看 |
| **审核通过** | 状态图标（勾）、提示文案「恭喜，审核通过！」 | 按钮「继续完成签约」→ `/apply-entry/contract` |
| **审核驳回** | 状态图标（叉）、驳回原因详情 | 按钮「返回修改」→ `/apply-entry`（回填数据） |

---

#### 合同签署 `/apply-entry/contract`

两步流程：

| 步骤 | 标题 | 内容 |
|------|------|------|
| 1 | 添加签约主体 | 统一社会信用代码输入、签约人姓名、签约人身份证号、签约人手机号 |
| 2 | 生成并签署合同 | 预览合同内容（PDF 渲染）、对接飞书电子签（或第三方电子签）完成签署 |

**交互**:
- 签约主体信息提交后生成合同
- 合同签署完成 → 跳转 `/apply-entry/qualification`

---

#### 资质授权 `/apply-entry/qualification`

| 区域 | 内容 |
|------|------|
| 说明 | 上传代理授权书的要求说明（格式、大小、模板下载） |
| 上传 | 文件拖拽/点击上传区域（支持 PDF、JPG、PNG） |
| 预览 | 已上传文件缩略图/文件名 |
| 模板 | 代理授权书模板下载链接 |

**交互**:
- 上传文件 → 点击「提交」
- 提交成功 → 入驻完成提示 → 按钮「进入工作台」→ `/dashboard`

---

### 4.3 工作台（对应飞书文档第四章）

#### 工作台首页 `/dashboard`

| 区域 | 内容 |
|------|------|
| 数据概览卡片 | 今日充值金额、今日新增用户、今日订单数、今日 DAU（四宫格卡片，含较昨日对比） |
| 趋势图 | 近 7 日 / 30 日充值金额趋势折线图 |
| 快捷操作 | 「生成推广链」「查看数据报表」「管理充值模板」快捷入口 |
| 待办提醒 | 未处理的审核通知、账单提醒、成员申请等 |

**权限**: 所有角色可见，但数据范围不同（集团看全量、主管看团队、投手看个人）

---

#### 短剧列表 `/dashboard/distribution/dramas`

| 区域 | 内容 |
|------|------|
| 筛选栏 | 搜索框（名称）、分类筛选（标签）、排序（热度/分佣/更新时间）、榜单切换（今日榜/飙升榜/新书榜） |
| 内容卡片网格 | 每张卡片：封面图、剧名、集数、分佣比例、热度指数、「生成推广链」按钮 |
| 分页 | 底部分页器 |

**交互**:
- 点击卡片 → 展开详情（简介、分集列表、推广素材下载）
- 点击「生成推广链」→ 弹窗选择回传规则 + 充值模板 → 生成链接 → 可在推广链管理中查看

**权限**: 所有角色可见

---

#### 网文列表 `/dashboard/distribution/novels`

与短剧列表结构一致，差异：

| 差异项 | 短剧 | 网文 |
|--------|------|------|
| 内容单位 | 集 | 章 |
| 卡片信息 | 集数、时长 | 章节数、字数 |
| 分类标签 | 都市、甜宠、悬疑等 | 都市、言情、玄幻、悬疑等 |

**权限**: 所有角色可见

---

#### 推广链管理 `/dashboard/distribution/promote`

| 区域 | 内容 |
|------|------|
| 操作栏 | 「新建推广链」按钮、搜索框（PID/内容名称） |
| 推广链表格 | 列：PID、关联内容（剧名/书名）、回传规则名称、充值模板名称、创建时间、累计点击、累计转化金额 |
| 行操作 | 复制链接、编辑、停用/启用、查看详情 |
| 分页 | 底部分页器 |

**交互**:
- 「新建推广链」→ 弹窗：选择内容 → 选择回传规则 → 选择充值模板 → 生成
- 「复制链接」→ 复制推广链接到剪贴板

**权限**: 集团、主管、投手

---

#### 广告回传 `/dashboard/distribution/postback`

| 区域 | 内容 |
|------|------|
| 操作栏 | 「新增回传规则」按钮 |
| 规则列表 | 列：规则名称、投放平台（巨量引擎/腾讯广告等）、转化事件类型、状态（启用/停用）、创建时间 |
| 行操作 | 编辑、停用/启用、删除 |

**新增/编辑回传规则弹窗**:

| 字段 | 说明 |
|------|------|
| 规则名称 | 文本输入 |
| 投放平台 | 下拉选择（巨量引擎、腾讯广告、百度推广等） |
| 转化事件映射 | 配置项列表：平台事件 ↔ 本平台事件的对应关系（如：巨量「付费」↔ 本平台「首充」） |
| 回传条件 | 回传触发条件（如：首充金额 ≥ X 元） |

**权限**: 集团、主管、投手

## 六、页面跳转流程

### 6.1 入驻主流程

```
落地页 (/)
  │
  ├─ 点击「登录/注册」───→ 登录注册 (/login)
  │                          │
  │                          ├─ 已入驻 ──→ 工作台 (/dashboard)
  │                          ├─ 未入驻 ──→ 信息录入 (/apply-entry)
  │                          ├─ 审核中 ──→ 审核状态 (/apply-entry/review)
  │                          ├─ 待签约 ──→ 合同签署 (/apply-entry/contract)
  │                          └─ 待授权 ──→ 资质授权 (/apply-entry/qualification)
  │
  └─ 点击「立即入驻」───→ (未登录→/login，已登录→/apply-entry)
```

### 6.2 入驻步骤流转

```
信息录入 (/apply-entry)
  │ 步骤 1 ↔ 2 ↔ 3 ↔ 4 ↔ 5 ↔ 6（可前进后退）
  │ 提交
  ▼
审核状态 (/apply-entry/review)
  ├─ 审核中 → 等待（可刷新）
  ├─ 驳回 → 返回信息录入（回填数据）
  └─ 通过 ↓
       ▼
合同签署 (/apply-entry/contract)
  │ 步骤 1: 添加签约主体
  │ 步骤 2: 生成并签署合同
  ▼
资质授权 (/apply-entry/qualification)
  │ 上传代理授权书 → 提交
  ▼
工作台 (/dashboard)  ✅ 入驻完成
```

### 6.3 工作台核心操作流

```
短剧/网文列表
  │ 选择内容 → 点击「生成推广链」
  │              │
  │              ├─ 选择回传规则（来自「广告回传」配置）
  │              ├─ 选择充值模板（来自「充值模板」配置）
  │              └─ 生成推广链
  ▼
推广链管理
  │ 查看/复制链接 → 投放到外部广告平台
  ▼
数据中心（查看投放效果）
  ├─ 应用统计：整体数据
  ├─ 推广统计：按 PID 维度
  ├─ 订单统计：订单明细
  └─ 用户统计：用户行为
```

### 6.4 结算流程

```
数据中心（确认数据）
  ▼
账单管理 (/dashboard/settlement/bills)
  │ 查看账单 → 确认账单
  ▼
提现管理 (/dashboard/settlement/withdraw)
  │ 发起提现 → 等待打款
```
